# Proyecto de Visión Artificial con YOLOv8 y OpenCV / Computer Vision Project with YOLOv8 and OpenCV

## Español

### Descripción

Este proyecto fue desarrollado utilizando YOLO en su versión 8 y OpenCV. Se descargaron 3000 imágenes del "Open Images Dataset", el modelo fue entrenado y validado utilizando un conjunto de datos etiquetado manualmente con LabelImg.

### Entrenamiento y Validación

Para el entrenamiento se utilizaron 2000 imágenes divididas en 5 clases: Persona, Naranja, Banana, Manzana y Pan. El conjunto de validación constó de 400 imágenes, representando el 20% de los datos.

### Rendimiento del Sistema
El sistema tiene un buen rendimiento durante la inferencia, aunque ocasionalmente se producen falsos positivos. Estos se pueden mitigar ajustando la confianza de las detecciones.

### Requisitos
- Python 3.8.10
- ultralytics 8.2.65
- opencv-python 4.10.0.84
- numpy 1.24.4

## English

### Description

This project was developed using YOLO version 8 and OpenCV. 3000 images were downloaded from the "Open Images Dataset", the model was trained and validated using a manually labeled dataset with LabelImg.

### Training and Validation
2000 images were used for training, divided into 5 classes: Person, Orange, Banana, Apple, and Bread. The validation set consisted of 400 images, representing 20% of the dataset.

### System Performance

The system performs well during inference, though occasional false positives occur. These can be mitigated by adjusting the detection confidence.

### Requirements
- Python 3.8.10
- ultralytics 8.2.65
- opencv-python 4.10.0.84
- numpy 1.24.4
